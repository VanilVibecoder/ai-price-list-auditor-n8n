- добавлен AI-маппинг столбцов через локальную Qwen 2.5 7B;
- добавлены JSON Schema и confidence gate;

- добавлены ошибки, предупреждения и классификация дублей;
- добавлена противоположная конвертация CSV ↔ XLSX;
- добавлен итоговый Telegram-отчёт;
- workflow очищен для публичной публикации.
